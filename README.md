# 🎨 CSS Motion Lab — Design Token Gallery

## 📌 Description
CSS Motion Lab is a collection of modern **CSS animations and UI motion effects** built using pure HTML and CSS.  
This project demonstrates how design tokens, animations, and responsive layouts can be combined to create visually engaging components.

---

## 🚀 Features
- 🎨 Theme switching (Void & Ember) using pure CSS
- 💫 Smooth animations using keyframes
- 🧩 Modular animation components
- 📱 Responsive grid layout using CSS Grid
- ⚡ No JavaScript used (Pure CSS project)

---

## 🧪 Animations Included
- Orbital Ring
- Morphing Blob
- Typewriter Effect
- Neon Flicker
- DNA Helix Animation
- Solar System Animation
- Equaliser Bars
- Spinner Trio
- Starburst Pulse
- Grid Flash Animation

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- CSS Grid  
- CSS Variables (Design Tokens)  
- Keyframe Animations  

---
